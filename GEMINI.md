# MOMA Engineering Mandates

## Design & Theme Compatibility
- **Dual-Theme Requirement:** ALL UI changes must fully support both Light and Dark modes.
- **CSS Variables:** Never use hardcoded hex values for colors. Always use the established CSS variables (e.g., `var(--surface)`, `var(--on-surface)`, `var(--primary)`) to ensure theme consistency.
- **Verification:** Always verify UI changes in both themes before finalizing a task.
